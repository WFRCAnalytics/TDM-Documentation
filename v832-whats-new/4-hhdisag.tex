Several changes were made to clean up and fix minor errors in the
household disaggregation and auto ownership scripts:

\begin{itemize}
\tightlist
\item
  Added Box Elder Vehicles to the totals calculated for the
  `VO\_Tmp\_CountyVehicleTotal.txt' file in the `3\_AutoOwnership.s'
  file.
\item
  Corrected VO\_DebugUtility.csv header row in `3\_AutoOwnership.s' to
  keep it from repeating. Commented out debug csv creation.
\item
  Corrected AVGINCOME in `Tmp\_Marginal\_Income\_beforeIPF.dbf'.
\item
  Commented out `test.csv' creation in
  `2\_ModelScripts\textbackslash1\_HHDisag\_AutoOwn\textbackslash1\_LifeCycle.s'.
\item
  Corrected CO\_TAZID in `LifeCycle\_Households\_Population.dbf' in
  `2\_ModelScripts\textbackslash1\_HHDisag\_AutoOwn\textbackslash1\_LifeCycle.s'.
\item
  Updated `Lookup - BYTAZAgePct - AllCo.csv' and source materials
  ('\_TAZ\_AgePct\_Lookup\_Source - 2020-06-23.xlsb') in
  `1\_Inputs\textbackslash0\_GlobalData\textbackslash1\_HHDisag\_AutoOwn'.
  This update reflects the updated source materials structure in the
  statewide spreadsheet and updated data for Summit/Wasatch counties and
  UDOT area. The Wasatch Front area data did not change.
\end{itemize}
